# Odin Recipes

A simple recipe website created as a first HTML project for [The Odin Project](https://www.theodinproject.com/).

## Description

This project is a basic website showcasing a collection o recipes. It was built to practice HTML skills learned, including:
- Basic HTML structure 
- Links and navigation between pages
- Ordered and unordered lists
- Images
- Text formatting

## Recipes included

- Recipe of Carbonara
- Recipe of Amatriciana
- Recipe of Tiramisu

## Technologies used

- HTML5
